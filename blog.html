What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is based event-bubbling method. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

How do you manipulate an element's attributes and styles using the DOM?

If you want to change other attributes like id , href , or src , you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set. The value of the attribute to set.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use methods like 'getElementById' to retrieve an element by its unique ID, or 'getElementsByClassName' and 'getElementsByTagName' to access multiple elements based on class or tag names. Additionally, 'querySelector' allows you to select the first element that matches a CSS selector, while 'querySelectorAll' retrieves all elements that match a given selector. These methods provide flexibility for accessing and manipulating elements within a web page.
