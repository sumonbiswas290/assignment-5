What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

JavaScript provides several methods to select DOM elements. Here are some of them:

  • Using getElementById:- let element = document.getElementById("myId");
  • Using getElementsByClassName:- let elements = document.getElementsByClassName("myClass");
  • Using getElementsByTagName:- let elements = document.getElementsByTagName("p");
  • Using querySelector:- let element = document.querySelector(".myClass");
  • Using querySelectorAll:- let elements = document.querySelectorAll(".myClass");

    • What is event delegation in the context of the DOM, and why is it useful?

    Event delegation is based event-bubbling method. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

    How do you manipulate an element's attributes and styles using the DOM?

    If you want to change other attributes like id , href , or src , you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set. The value of the attribute to set.
